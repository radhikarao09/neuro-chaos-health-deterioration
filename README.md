# 🧠 Neuro-Chaos Health Deterioration Score Prediction

## 📌 Overview
This project predicts the **risk of patient health deterioration** using vital signs and machine learning techniques.  
It uses a trained regression model to analyze physiological parameters and estimate a deterioration score.

---

## 🚀 Features
- Predicts patient deterioration risk
- User-friendly interface using Streamlit
- Uses real-time input of vital signs
- Machine Learning model for accurate prediction

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

---

## 📊 Input Parameters
The model takes the following inputs:
- Age
- Gender
- Heart Rate
- Respiratory Rate
- Body Temperature
- Oxygen Saturation (SpO2)
- Systolic Blood Pressure
- Diastolic Blood Pressure
- Weight
- Height
- Heart Rate Variability (HRV)
- Pulse Pressure
- Body Mass Index (BMI)
